# smarter_resume
## Usage

Clone app in your local

## Install Dependencies

```
# project_folder/frontend
npm install

# project_folder/backend
npm install
```

## Run backend

```
# Run server
npm start

# Run Frontend
npm start
```

## Run backend test 

```
# Run unit test
npm run test


## Access

```
 # Access the web app in browser:
 http://localhost:3000
 
 # Access the Server:
 http://localhost:5000/
```
